Research Objectives

The main objective of this research is to build an event-based system which can detect malicious URLs early in the attack lifecycle. In doing so, we want to increase accuracy of the Malicious URL Classifier by using ensemble modeling and achieve an higher perfomance using feature reduction. Scalability of the model and system will be build from the core to handle various volumes of URLs.

About the Project

The Event-Driven Malicious URL Extractor is presented with the primary aim of addressing traditional system limitations and offering an interface for users to protect themselves by detecting phishing/malicious URLs in real time. Using user-input event-related keywords and leveraging NLP (Natural Language Processing) algorithms to match them with the accompanying URL (Uniform Resource Locator) token data to determine whether the URLs are malicious or benign. We further offer a browser extension to automate the process of blocking malicious URLs from an endpoint device
